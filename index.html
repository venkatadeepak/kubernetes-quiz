Kubernetes Quiz

1. What is Kubernetes?

A) A programming language
B) A container orchestration platform
C) A database management system
D) A web server
Kubernetes is a container orchestration platform that automates the deployment, scaling, and management of containerized applications.

2. What is a Pod in Kubernetes?

A) A single container
B) The smallest deployable unit that can contain one or more containers
C) A storage volume
D) A network interface
A Pod is the smallest deployable unit in Kubernetes that can contain one or more containers sharing the same network namespace and storage volumes.

3. What is a Kubernetes Service?

A) A web service running in a container
B) A load balancer
C) An abstraction that defines a logical set of Pods and a policy to access them
D) A monitoring tool
A Service is an abstraction that defines a logical set of Pods and a policy by which to access them, enabling loose coupling between dependent Pods.

4. What is a Deployment in Kubernetes?

A) The process of installing Kubernetes
B) A declarative way to manage a set of Pods and ReplicaSets
C) A way to deploy containers directly
D) A networking configuration
A Deployment is a declarative way to manage a set of Pods and ReplicaSets, providing declarative updates for Pods and ReplicaSets.

5. What is a ConfigMap?

A) A map of cluster nodes
B) A storage object for sensitive data
C) An API object used to store non-confidential data in key-value pairs
D) A network routing table
ConfigMap is an API object used to store non-confidential data in key-value pairs, allowing you to decouple configuration artifacts from image content.

6. What is a Namespace in Kubernetes?

A) A way to organize objects in a cluster
B) A type of container
C) A network protocol
D) A storage volume
A Namespace is a way to organize objects in a cluster, providing a scope for names and allowing you to divide cluster resources between multiple users.

7. What is a StatefulSet?

A) A type of persistent storage
B) A workload API object used to manage stateful applications
C) A database configuration
D) A network state manager
StatefulSet is a workload API object used to manage stateful applications, providing guarantees about the ordering and uniqueness of Pods.

8. What is an Ingress?

A) A network entry point
B) An API object that manages external access to services in a cluster
C) A storage system
D) A container runtime
Ingress is an API object that manages external access to services in a cluster, typically providing HTTP routing rules.

9. What is a DaemonSet?

A) A background process
B) A type of container
C) A workload that ensures all nodes run a copy of a Pod
D) A monitoring system
A DaemonSet ensures that all (or some) nodes run a copy of a Pod, useful for running cluster-wide services like logging or monitoring agents.

10. What is a PersistentVolume?

A) A piece of storage in the cluster
B) A type of container storage
C) A backup system
D) A memory management system
A PersistentVolume is a piece of storage in the cluster that has been provisioned by an administrator or dynamically provisioned using Storage Classes.
